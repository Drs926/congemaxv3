# Function Map - Cong√©Max V3

## Objectif
Aligner les modules fonctionnels sur les 8 ecrans contractuels V3.

## Modules principaux
1. Demarrage
- Ecran: Splash.
- Capacite: determiner la route de demarrage.
- Acces: Freemium.

2. Profil utilisateur
- Ecran: Onboarding_Profil.
- Capacite: enregistrer locale, timezone, semaine de debut.
- Acces: Freemium.

3. Convention active
- Ecran: Onboarding_Convention.
- Capacite: activer une convention unique, reinitialiser scenarios/resultats en cas de changement.
- Acces: Freemium.

4. Vue synthese
- Ecran: TableauDeBord.
- Capacite: consulter soldes et entree vers planification/simulation/resultats/parametres.
- Acces: Freemium.

5. Planification
- Ecran: Calendrier_Planification.
- Capacite: saisir plage de dates candidate.
- Acces: Freemium.

6. Simulation deterministe
- Ecran: Simulation_Config.
- Capacite: configurer scenario puis lancer un calcul deterministe.
- Acces: Freemium.

7. Resultats et commentaire
- Ecran: Resultats.
- Capacite: afficher metriques deterministes, demander commentaire IA readonly.
- Acces: Freemium.

8. Optimisation annuelle
- Ecran: Resultats.
- Capacite: lancer optimisation annuelle.
- Acces: Premium.
- Fallback Freemium: CTA `Decouvrir Premium` vers Parametres.

9. Parametres et offre premium
- Ecran: Parametres.
- Capacite: gerer reglage local, statut premium, changement de convention active.
- Acces: Freemium.
